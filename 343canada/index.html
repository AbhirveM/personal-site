About

343Canada is an election forecasting model for Canadian federal elections. It aggregates polling data, models uncertainty using Monte Carlo simulations, and generates riding-level predictions with geographic visualization.


Built to explore statistical modeling and geospatial data visualization, the model weighs pollsters based on historical accuracy and produces probability distributions for seat counts and election outcomes.

Tech Stack

Python Core language for data processing and model implementation
NumPy / SciPy Statistical computations and Monte Carlo simulations
GeoPandas GIS data handling and electoral map visualization
Matplotlib Probability distributions and forecast charts

Key Features

  • Poll aggregation with pollster weighting based on historical accuracy
  • Monte Carlo simulation for seat projections and outcome probabilities
  • Riding-level visualization with interactive maps
  • Uncertainty quantification with confidence intervals
  • Historical data integration for model calibration
